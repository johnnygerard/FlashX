Privacy policy

Last updated October 20, 2022

Means of data storage

This website uses MongoDB Atlas to store user data.
The database infrastructure is supplied by Microsoft Azure.

Data collected

  • Account creation date and time
  • Password hash (read Additional notes for more info)

Data sent by the user

  • Username
  • Flashcard collections

Data residency

All user data is stored in a GDPR-compliant database located in the Netherlands.

Backup and data replication

Database backup is automated: up to 2 snapshots, one every 6 hours.

Another feature of MongoDB Atlas is data replication.
Similar to data backup but instead provides server availability and data disaster recovery.
For example if a database server is unavailable or damaged, the replicated servers can be used as a fallback.

Additional notes

The user password is not stored anywhere and consequently cannot be recovered.
The password hash is computed once at account creation time or when a password is updated.
It is used to authenticate the user at each login attempt.
